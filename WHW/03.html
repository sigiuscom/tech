Проверка списка HSTS

Проверка списка HSTS

Браузер проверяет свой "предзагруженный" список HSTS (HTTP Strict Transport Security, строгая безопасность транспорта http). Это список вебсайтов, который запросили, чтоб с ними связывались только по https. Если сайт в этом списке, браузер будет посылать запросы по https вместо http. В противном случае начальный запрос будет послан по http. Вебсайт также может использовать политику HSTS без того, чтоб быть в списке HSTS. В первый http запрос к этому вебсайту пользователь получит ответ, запрашивающий, что пользователь должен посылать запросы только по https. Однако, этот единственный запрос по http может потенциально оставить пользователя уязвимым к атаке понижения протокола, вот почему список HSTS включен во все современные браузеры
